# PROJECT REPORT


This Digital Literacy project helped me understand the importance of using digital tools effectively and responsibly. In Task 1, I created an infographic using Canva that explained digital literacy, useful tools, safe internet practices, and email etiquette. This improved my ability to present information visually.

In Task 2, I built my digital portfolio using GitHub, LinkedIn, and Kaggle. These platforms helped me understand how to create a professional online presence and how they can support my career growth over the next four years.

In Task 3, I practiced coding on HackerRank and created a Google Form quiz. This enhanced my problem-solving skills and showed me how digital tools can support academic learning and collaboration.

In Task 4, I learned professional email writing and understood the importance of proper communication. Poor email etiquette can lead to misunderstandings, so using a clear structure and polite tone is essential.

In Task 5, I studied UPI fraud and was surprised by how easily people can be tricked. I learned to be more cautious online and always verify before making any transactions. Overall, this project improved my digital awareness, communication skills, and preparedness for the future.
